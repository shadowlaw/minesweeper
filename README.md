# Minesweeper 0.1.0

A Simple desktop minesweeper game written in java Java 1.8.

## How to run
- Install the Java JRE onto your system (version 8 or higher)
- Download the latest .jar from the [releases](https://github.com/shadowlaw/minesweeper/releases) list 
- Run the application using the command `java -jar minesweeper.jar`

## Features not yet implemented
- Chording
- Medium, Hard and Custom game modes.
